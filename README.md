# Egor academic webpage (GitHub Pages)

This is a minimal multi-page site implemented with plain HTML/CSS/JS.

## Structure
- `index.html` — Home
- `research.html`
- `cv.html` (+ `assets/cv.pdf`)
- `teaching.html`
- `activities.html`
- `this-and-that.html`
- `assets/styles.css`
- `assets/main.js`

## Deploy to GitHub Pages
1. Create a repo (e.g. `yourname.github.io` or any repo name).
2. Upload all files **to the root of the repo** (keep the `assets/` folder).
3. In GitHub: Settings → Pages → Deploy from branch → `main` / root.
4. Open the provided URL.

## Edit content
Open any `.html` file and change text in the `<article>` section.
